## 보수법칙

$$A + \overline{A} = 1$$
$$A \cdot \overline{A} = 0$$


## 분배법칙

$$A(B + C) = AB + AC$$
$$A + BC = (A + B)(A + C)$$


## 흡수법칙
$$A + AB = A$$
$$A(A + B) = A$$

예:
$$A + A\overline{B} = A$$
$$A(A + \overline{B}) = A$$


## 드모르간 정리

$$\overline{A + B} = \overline{A} + \overline{B}$$
$$\overline{AB} = \overline{A} + \overline{B}$$


##  XOR 관련 핵심 패턴

$$A \oplus B = \overline{A}B + A\overline{B}$$

$$= (A + B)(\overline{A} + \overline{B})$$

➡️ *"다르다"를 표현하는 연산*
