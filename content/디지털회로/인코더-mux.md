title: 인코더, mux
---
## 인코더(Encoder)
인코더는 2^n 개의 입력 신호 중에서 하나의 입력 신호를 선택하여 n 비트의 이진 코드로 변환하는 조합 논리 회로입니다. 예를 들어, 4-to-2 인코더는 4개의 입력 신호를 받아 2비트의 이진 코드로 출력합니다.

## 디코더(Decoder)
디코더는 n 비트의 이진 코드를 받아 2^n 개의 출력 신호 중 하나를 활성화하는 조합 논리 회로입니다. 예를 들어, 2-to-4 디코더는 2비트의 이진 코드를 받아 4개의 출력 신호 중 하나를 활성화합니다.

## 멀티플렉서(Multiplexer, MUX)
멀티플렉서는 여러 개의 입력 신호 중에서 하나를 선택하여 출력하는 조합 논리 회로입니다. n-to-1 멀티플렉서는 2^n개의 입력 신호와 n개의 선택 신호를 사용하여 하나의 출력 신호를 생성합니다. 예를 들어, 4-to-1 멀티플렉서는 4개의 입력 신호와 2개의 선택 신호를 사용하여 하나의 출력 신호를 생성합니다.

## 디멀티플렉서(Demultiplexer, DEMUX)
디멀티플렉서는 하나의 입력 신호를 여러 개의 출력 신호 중 하나로 분배하는 조합 논리 회로입니다. 1-to-n 디멀티플렉서는 하나의 입력 신호와 n개의 선택 신호를 사용하여 2^n개의 출력 신호 중 하나를 활성화합니다. 예를 들어, 1-to-4 디멀티플렉서는 하나의 입력 신호와 2개의 선택 신호를 사용하여 4개의 출력 신호 중 하나를 활성화합니다.
