## 주제
즐거운 배포

## 팀 주제
슬랙봇을 사용한 CI/CD 자동화

## 사용 기술
GCP Cloud Deploy, N8N, Slack API 등등

## 내 역할
N8n 파이프라인을 구성하고 Cloud Deploy REST API를 사용하여 N8N 노드에서 API 호출이 가능하게 한다

## 결과물
- N8N 노드에서 Cloud Deploy REST API를 호출하여 슬랙봇으로 카나리 배포를 가능하게 함
- 슬랙봇으로 배포 상태를 확인 가능하게 함
