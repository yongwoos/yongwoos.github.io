---
title: map
---
`map()` 함수는 Python의 내장 함수로, 주어진 함수를 반복 가능한 객체의 각 요소에 적용하여 새로운 이터레이터를 반환합니다. `map()` 함수는 다음과 같은 형식으로 사용됩니다.

`map(function, iterable, ...)`여기서 function은 각 요소에 적용할 함수이고, iterable은 반복 가능한 객체입니다. `map()` 함수는 여러 개의 iterable을 동시에 처리할 수도 있습니다. 예를 들어,

```python
map(lambda x, y: x + y, [1, 2, 3], [4, 5, 6])
```
을 사용하면 `[5, 7, 9]`가 반환됩니다.

또한, `map()` 함수는 여러 개의 함수를 적용할 수도 있습니다. 예를 들어, `map(str.upper, sys.stdin.readline().split())`을 사용하면 입력받은 문자열의 각 단어를 대문자로 변환할 수 있습니다.

map 반환 타입은 map 객체입니다. map 객체는 이터레이터로, 각 요소에 대해 지정된 함수를 적용한 결과를 순차적으로 반환합니다. 예를 들어, `map(int, ["1", "2", "3"])`을 사용하면 map 객체가 생성되고, 이 객체를 리스트로 변환하면 [1, 2, 3]이 됩니다. map 객체는 메모리를 효율적으로 사용하며, 필요한 경우에만 요소를 계산하기 때문에 대용량 데이터 처리에 유용합니다. 또한, map 객체는 반복문에서 사용할 수 있으며, 필요한 경우에만 요소를 계산하여 처리할 수 있습니다.
