---

## 스프레드(Spread) 개요

채권의 스프레드는 **특정 채권의 수익률이 기준 수익률보다 얼마나 높은가**를 나타내는 지표입니다. 신용위험·유동성위험 등을 수익률 차이로 표현합니다.

---

## 1. T스프레드 (Treasury Spread / Nominal Spread)

**정의:** 채권의 YTM에서 만기가 동일한 국채(Treasury) YTM을 단순 차감한 값

$$T\text{-spread} = YTM_{\text{채권}} - YTM_{\text{국채(동일만기)}}$$

**특징:**
- 계산이 가장 단순
- 단 하나의 기준 포인트(만기 매칭 국채)만 사용
- **수익률 곡선의 형태를 전혀 반영하지 않음** → 이게 핵심 단점

**단점 예시:** 5년 회사채 vs 5년 국채 YTM 차이만 보기 때문에, 중간 현금흐름(쿠폰)이 상이한 구조를 무시함

---

## 2. Z스프레드 (Zero-Volatility Spread)

**정의:** 현물금리(spot rate) 기반 국채 수익률 곡선의 **각 시점 금리에 일정한 스프레드(Z)를 더해서** 채권의 모든 현금흐름을 할인했을 때 시장가격과 일치하도록 만드는 상수

$$P = \sum_{t} \frac{CF_t}{(1 + r_t + Z)^t}$$

- $r_t$: t시점의 무위험 현물금리(spot rate)
- $Z$: Z스프레드 (전 기간에 걸쳐 동일하게 적용)

**특징:**
- **수익률 곡선 전체**를 활용 → T스프레드보다 정교함
- 쿠폰 지급 시점마다 각각 해당 spot rate 적용
- 옵션이 없는 채권에 적합 (옵션 내재 채권에는 OAS 사용)
- "Zero-Volatility"라는 이름은 **금리 변동성을 0으로 가정**한다는 의미

**T스프레드와의 차이:**

| | T스프레드 | Z스프레드 |
|---|---|---|
| 기준 | 단일 만기 국채 YTM | 전체 spot rate 곡선 |
| 현금흐름 반영 | ✗ | ✓ |
| 정교함 | 낮음 | 높음 |

---

## 3. CDS 스프레드 (Credit Default Swap Spread)

**정의:** 신용부도스왑에서 **보장 매입자(protection buyer)가 보장 매도자에게 지급하는 연간 프리미엄**으로, 특정 준거 기업(reference entity)의 **신용위험**을 직접적으로 나타냄

```
보장 매입자 ──[CDS 스프레드(프리미엄)]──▶ 보장 매도자
보장 매입자 ◀──[부도 시 손실 보전]────── 보장 매도자
```

**특징:**
- 채권을 실제로 보유하지 않아도 신용위험 거래 가능
- 시장에서 실시간으로 형성 → **시장 참여자의 신용위험 인식**을 직접 반영
- 단위: bp(베이시스포인트), 연간 기준
- CDS 스프레드 ↑ → 시장이 해당 기업/국가의 부도 가능성을 높게 봄

**채권 스프레드와의 관계 (CDS-Bond Basis):**

이론적으로 CDS 스프레드 ≈ Z스프레드여야 하지만, 실제로는 차이가 발생합니다.

$$\text{CDS-Bond Basis} = \text{CDS 스프레드} - \text{Z스프레드}$$

- **Basis > 0:** CDS가 더 비쌈 (위기 시 일반적)
- **Basis < 0:** 채권이 상대적으로 저평가 → 차익거래 기회

## 세 가지 비교 요약

| | T스프레드 | Z스프레드 | CDS스프레드 |
|---|---|---|---|
| **측정 대상** | 채권 신용·유동성 위험 | 채권 신용·유동성 위험 | 순수 신용위험 |
| **기준** | 단일 국채 YTM | Spot rate 곡선 전체 | CDS 시장 프리미엄 |
| **시장** | 채권시장 | 채권시장 | 파생상품시장 |
| **금리 변동성 반영** | ✗ | ✗ | ✗ (OAS는 반영) |
| **옵션 반영** | ✗ | ✗ | ✗ |
| **정교함** | 낮음 | 중간 | (별도 차원) |

> **실무 포인트:** 옵션부 채권(callable bond 등)에는 Z스프레드 대신 **OAS(Option-Adjusted Spread)** 를 사용합니다. OAS = Z스프레드 − 옵션 가치.

## T스프레드의 단점 — 풀어서 설명

### 먼저 배경: 채권 가격은 어떻게 결정되나?

채권의 현금흐름(쿠폰 + 원금)은 **여러 시점에 걸쳐** 발생합니다. 예를 들어 5년 만기 채권이라면:

```
1년 후 쿠폰 / 2년 후 쿠폰 / 3년 후 쿠폰 / 4년 후 쿠폰 / 5년 후 쿠폰+원금
```

각 시점의 현금흐름은 **그 시점에 맞는 금리**로 할인해야 이론적으로 정확합니다.

---

### T스프레드가 뭘 하냐?

5년 만기 회사채라면, **5년 국채 YTM** 하나만 떼다가 비교합니다.

$$T\text{-spread} = YTM_{\text{회사채, 5년}} - YTM_{\text{국채, 5년}}$$

그런데 YTM이라는 숫자 자체가 이미 **모든 현금흐름을 하나의 단일 할인율로 뭉개버린 값**입니다.

---

### 문제가 뭐냐?

실제 무위험 금리는 만기마다 다릅니다. 예를 들어:

| 만기 | 국채 Spot Rate |
|------|--------------|
| 1년  | 3.0% |
| 2년  | 3.5% |
| 3년  | 4.0% |
| 4년  | 4.3% |
| 5년  | 4.5% |

이게 **수익률 곡선의 형태**입니다.

T스프레드는 이 곡선을 전혀 안 보고, 그냥 **"5년 국채 YTM = 4.5%"** 하나만 씁니다. 1년 후 쿠폰을 할인할 때도, 3년 후 쿠폰을 할인할 때도, 전부 4.5% 하나로 퉁칩니다.

---

### 그래서 뭐가 문제냐?

두 채권이 **만기는 같아도 쿠폰 구조가 다르면**, T스프레드 비교가 왜곡됩니다.

- **고쿠폰 채권:** 초기(단기) 현금흐름 비중이 큼 → 단기 spot rate의 영향을 더 많이 받아야 함
- **저쿠폰 채권:** 만기 원금 비중이 큼 → 장기 spot rate의 영향을 더 많이 받아야 함

근데 T스프레드는 둘 다 "5년 국채 YTM" 하나로 비교하니까, **수익률 곡선이 우상향이냐 역전이냐에 따라 스프레드가 왜곡**됩니다.

---

### 한 줄 요약

> T스프레드는 **"5년짜리"라는 딱지 하나**만 보고 비교하는 거라, 그 5년 동안 현금흐름이 어느 시점에 얼마나 오는지를 무시한다. Z스프레드는 각 쿠폰 시점마다 그 시점에 맞는 spot rate를 적용해서 이 문제를 해결한 것.

## 정확히 짚기

**쿠폰(이자)은 맞아요, 처음부터 고정입니다.**

예: 액면 1000원, 쿠폰 5% → 매년 50원씩, 이건 변하지 않음.

---

## 그럼 뭐가 다르냐?

다른 건 **쿠폰 금액**이 아니라, 그 쿠폰을 **할인하는 금리**입니다.

1년 후에 받는 50원과 4년 후에 받는 50원은 **같은 금액이지만, 현재가치로 환산할 때 다른 금리를 써야 한다**는 거예요.

$$PV = \frac{50}{(1+r_1)^1} + \frac{50}{(1+r_2)^2} + \frac{50}{(1+r_3)^3} + \frac{50}{(1+r_4)^4} + \frac{1050}{(1+r_5)^5}$$

여기서 $r_1, r_2, r_3, r_4, r_5$가 각 시점의 **spot rate**이고, 이게 전부 다릅니다.

---

## T스프레드의 문제

T스프레드는 이걸 무시하고 $r_1 = r_2 = r_3 = r_4 = r_5 = 4.5\%$ (5년 국채 YTM 하나)로 퉁칩니다.

Z스프레드는 각 시점마다 제대로 된 spot rate를 써서 할인합니다.

---

**한 줄 요약:**
> 쿠폰 금액이 다른 게 아니라, **시점마다 적용해야 할 무위험 할인율이 다른데** T스프레드는 그걸 무시한다는 것.

네, 채권 가격 자체가 **미래 현금흐름의 현재가치 합**으로 정의됩니다.

$$P = \frac{C}{(1+r)^1} + \frac{C}{(1+r)^2} + \cdots + \frac{C+F}{(1+r)^n}$$

이게 채권 가격의 정의 자체예요.

---

그러니까 흐름을 다시 정리하면:

1. 채권 가격 = 현금흐름을 할인한 현재가치의 합
2. 할인율로 뭘 쓰냐가 문제
3. **T스프레드:** 5년 국채 YTM 하나로 전부 할인 (단순하지만 부정확)
4. **Z스프레드:** 각 시점의 spot rate로 따로따로 할인 (더 정확)

Z스프레드에서 "Z를 더한다"는 게 결국:

$$P = \frac{C}{(1+r_1+Z)^1} + \frac{C}{(1+r_2+Z)^2} + \cdots + \frac{C+F}{(1+r_n+Z)^n}$$

이 식이 시장가격 $P$와 일치하도록 $Z$를 찾는 것입니다. 그 $Z$가 Z스프레드예요.



그러면 반대로, **은행 금리가 10%인 세상**에서 "3년 후 105만원을 주는 채권"을 지금 100만원에 사겠어요?

안 사겠죠. 은행에 넣으면 133만원이 되는데.

그러면 이 채권, 지금 얼마에 팔려야 사람들이 살까요?

즉, 시장금리만큼 채권 가격은 할인된다

$$P = \frac{105}{(1.1)^3}$$

이게 할인이에요. **미래에 받을 돈을, 지금 시장 금리 기준으로 역산한 현재가치.**

채권 가격은 이렇게 결정돼요. 액면가 100만원에 팔리는 게 아니라, 시장 금리에 따라 가격이 올라가고 내려가요. 그래서 할인을 하는 거예요.

> 채권 가격에 시장금리만큼 곱한게 원금+이자랑 같아야 한다.
>
>1년후 받는 이자 50원은 1년시장금리만큼 할인, 2년후 받는 50원은 2년 시장금리만큼 할인.... 이렇게 마지막 3년째 받는 이자+원금은 3년 시장금리만큼 할인한거 다 더한게 채권 가격이 된다.



$$P = \frac{50}{(1+r_1)^1} + \frac{50}{(1+r_2)^2} + \frac{1050}{(1+r_3)^3}$$

이게 채권 가격의 본질이고, Z스프레드는 여기서 각 $r_t$에 동일한 $Z$를 더했을 때 이 식이 시장가격과 일치하는 $Z$를 찾는 거예요.

$$1050 = \frac{50}{(1+r_1+Z)^1} + \frac{50}{(1+r_2+Z)^2} + \frac{1050}{(1+r_3+Z)^3}$$

그 $Z$가 곧 "국채 대비 이 회사채가 추가로 요구하는 수익률", 즉 신용위험 + 유동성위험의 크기예요.
