---
title: 크로스 커런시 베이시스
---
크로스 커런시 베이시스(CCY basis)는 한마디로


**“이론적으로 맞아야 할 금리차(CIP)가 실제 시장에서 깨지면서 생기는 추가 스프레드”**

## 1. 기준: **CIP (Covered Interest Parity)**

이론적으로는 아래가 성립해야 한다:

$$
(1 + r_{KRW}) = \frac{F}{S} (1 + r_{USD})
$$

* $S$: 현물환율
* $F$: 선물환율
* $r_{KRW}$, $r_{USD}$: 각 통화 금리

즉
👉 **선물환 가격 = 금리차로 완전히 설명됨**


## 2. 현실: CIP 깨짐 → 베이시스 발생

실제 시장에서는:

$$
(1 + r_{KRW}) \neq \frac{F}{S} (1 + r_{USD})
$$

그래서 보정항을 넣는다:

$$
(1 + r_{KRW}) = \frac{F}{S} (1 + r_{USD} + \text{basis})
$$

여기서
👉 **basis = 크로스 커런시 베이시스**


## 3. 직관 (제일 중요)

### USD 기준으로 설명

* **basis < 0 (음수)**
  → “달러 조달이 비싸다”
  → 달러 빌리려면 금리 더 얹어줘야 함

* **basis > 0 (양수)**
  → “달러 조달이 싸다”


## 4. 왜 생기냐 (이론 말고 실제 이유)

CIP는 “완벽한 차익거래 가능” 가정인데 현실은 다름:

### (1) 달러 수요 폭발

* 글로벌 금융은 달러 중심
* 위기 때 전 세계가 USD 필요
  → 달러 funding 프리미엄 발생

### (2) 은행 밸런스시트 제약

* 레버리지 규제 (SLR 등)
* 차익거래 하고 싶어도 못함

### (3) 신용 리스크

* 통화마다 크레딧 다름
* 동일 금리라도 체감 비용 다름

### (4) 수급 불균형

* 예: 한국 기관 → 달러 필요 (해외투자)
* 반대편 부족

## 5. 거래 형태로 보면

크로스 커런시 베이시스는 실제로는
👉 **Cross Currency Swap (CCS)** 시장에서 관측됨

구조:

* 원화 금리 + α
* vs 달러 금리

여기서 α가 바로 basis

## 6. FX swap / forward 연결

CIP 깨지면 이런 일이 발생:

### (정상)

* 현물 + 선물 → 금리차만 반영

### (깨짐)

* 선물환 가격이 금리차 이상/이하로 움직임
  → **추가 수익 기회 발생**

그래서:

* FX swap
* 선물환
* CCS

이 세 개가 전부
👉 **같은 현상 (달러 funding 문제)**를 다른 형태로 표현


## 7. 핵심 요약 (중요 포인트만)

* 크로스 커런시 베이시스 = CIP 깨짐의 크기
* 본질 = **달러 조달 프리미엄**
* 음수 베이시스 = 달러 부족
* 원인 = 규제 + 수급 + 글로벌 달러 의존
* 관측 = CCS / FX swap / forward 시장

## 8. 한 줄로 정리

👉 **“금리차만으로 설명 안 되는 환율 괴리 = 달러 자금 시장 스트레스”**
