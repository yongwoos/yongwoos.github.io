### 저역통과필터
RC 필터에서 가장 기본적인 형태는 저역통과필터입니다. 입력 신호가 저주파일 때는 거의 그대로 출력으로 전달되고, 고주파일 때는 출력이 감소하는 특성을 가집니다.

RL 필터도 저역통과 특성을 가질 수 있습니다. 인덕터가 주파수가 높아질수록 임피던스가 증가하기 때문에 고주파를 차단하는 역할을 합니다.

#### RC 필터에서 **C 앞에 직렬 저항 R이 붙는 이유
>주파수에 따라 전압 분배가 일어나게 만들기 위해서

핵심은 **커패시터의 임피던스가 주파수에 따라 변한다**는 점이야.

커패시터 임피던스
[
Z_C = \frac{1}{j\omega C}
]

* **저주파** → ( \omega ) 작음 → (Z_C) 큼 (거의 open)
* **고주파** → ( \omega ) 큼 → (Z_C) 작음 (거의 short)

여기에 **저항 R을 직렬로 두면 R과 C가 전압 분배기 역할**을 하게 돼.

예를 들어 가장 흔한 **RC 저역통과 필터 (Low-pass)**

입력 → R → 출력 → C → GND

출력은 C 앞에서 뽑음.

주파수별 동작

**1️⃣ 저주파**

* (Z_C) 매우 큼
* 전류 거의 안 흐름
* R에 전압 강하 거의 없음

그래서
입력 ≈ 출력

→ **저주파 통과**

**2️⃣ 고주파**

* (Z_C) 매우 작음
* 전류 많이 흐름
* R에서 전압 떨어짐

결과
출력이 작아짐

→ **고주파 차단**

그래서 전달함수도

[
H(j\omega)=\frac{1}{1+j\omega RC}
]

컷오프 주파수

[
f_c = \frac{1}{2\pi RC}
]

---

### 만약 R 없이 C만 있으면?

입력 → C → GND

이건 **필터가 아니라 그냥 커패시터 하나 달린 회로**라서
출력 전압이 정의되지 않아.

즉

* **전압 분배가 안 생김**
* **주파수 선택성이 안 만들어짐**

그래서 **반드시 R이 필요**해.

---

한 줄 핵심

**RC 필터의 R은 “주파수에 따라 변하는 C와 전압 분배를 만들어서 필터 특성을 만들기 위해 존재한다.”**
