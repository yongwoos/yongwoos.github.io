| 회로                  | 입력  | 출력  | 공통 단자 |
| ------------------- | --- | --- | ----- |
| 공통 이미터(CE)          | 베이스 | 컬렉터 | 이미터   |
| 공통 컬렉터(CC, 에미터 팔로워) | 베이스 | 이미터 | 컬렉터   |
| 공통 베이스(CB)          | 이미터 | 컬렉터 | 베이스   |

### 1. 공통 이미터(CE)

* 입력: 베이스
* 출력: 컬렉터
* 공통: 이미터(보통 GND)

```
입력 → B
       │
       │
      Q
       │
출력 ← C
       │
      Rc
       │
      VCC

E → GND (공통)
```

특징

* 전압이득 큼
* 전류이득 큼
* **출력이 입력과 180° 반전**

### 2. 공통 컬렉터(CC, Emitter Follower)

* 입력: 베이스
* 출력: 이미터
* 공통: 컬렉터(VCC에 연결되는 경우가 많음)

```
입력 → B
       │
       │
      Q
       │
출력 ← E
       │
      Re
       │
     GND

C → VCC (공통)
```

특징

* 전압이득 ≈ 1
* 전류이득 큼
* 위상 반전 없음
* 버퍼로 많이 사용

### 3. 공통 베이스(CB)

* 입력: 이미터
* 출력: 컬렉터
* 공통: 베이스

특징

* 전압이득 큼
* 전류이득 ≈ 1
* 고주파 회로에서 사용

### 왜 이름이 '공통'인가?

공통은 **입력 회로와 출력 회로가 함께 사용하는 단자**를 의미한다.

* 공통 이미터 → 입력(B-E), 출력(C-E) 모두 **이미터**를 공유
* 공통 컬렉터 → 입력(B-C), 출력(E-C) 모두 **컬렉터**를 공유
* 공통 베이스 → 입력(E-B), 출력(C-B) 모두 **베이스**를 공유

따라서 질문한 내용은 맞다.

* **공통 이미터(CE): 베이스 입력 → 컬렉터 출력**
* **공통 컬렉터(CC): 베이스 입력 → 이미터 출력**

이 두 형태가 실제 회로에서 가장 많이 사용된다.
