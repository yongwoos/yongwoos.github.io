---
title: PLL(위상 동기 루프)
---
> **PLL은 기준 입력 신호의 위상·주파수에 출력 신호를 동기화시키는 피드백 제어 회로**

## 왜 필요한가

* **주파수 생성 / 배수**
* **클럭 동기화**
* **통신 신호 복조 및 클럭 복구**
* **잡음에 강한 안정적인 클럭 생성**

## 기본 구성 요소

### (1) 기준신호 (Reference Input)

* 외부에서 들어오는 기준 클럭
* PLL이 **따라가야 할 목표**


### (2) 위상 검출기 (Phase Detector, PFD)

* 기준신호와 피드백 신호의 **위상 차이** 검출
* 위상 차 → 전압/펄스 형태의 오차 신호 생성



### (3) 루프 필터 (Loop Filter)

* 위상 검출기 출력의 고주파 성분 제거
* **안정적인 제어 전압** 생성
* PLL의 **안정도·응답속도** 결정


### (4) VCO (Voltage-Controlled Oscillator)

* 제어 전압에 따라 주파수가 변하는 발진기
* **PLL 출력 신호의 근원**


### (5) 분주기 (Divider)

* VCO 출력을 **1/N로 나눔**
* 기준신호와 **같은 주파수 영역에서 비교** 가능하게 함

## 4️⃣ 동작 원리 (흐름)

```
기준신호
   ↓
[ 위상검출기 ] ←── VCO 출력 ÷ N
   ↓
[ 루프필터 ]
   ↓
[   VCO   ] ──→ 출력
```

1. 기준신호와 분주된 출력의 위상 비교
2. 위상 차 → 오차 신호 발생
3. 오차 신호로 VCO 주파수 조정
4. 위상 차가 일정해지면 **Lock**

## 5️⃣ Lock 상태에서의 관계식

[
\frac{f_{out}}{N} = f_{ref}
]

[
f_{out} = N \cdot f_{ref}
]

* N=1 → 입력과 출력 주파수 동일
* N≠1 → 주파수 배수 가능

---

## 6️⃣ 입력과 출력의 관계 (중요)

| 항목    | 같아짐? | 설명                 |
| ----- | ---- | ------------------ |
| 주파수   | ✅    | (또는 N배)            |
| 위상    | ⚠️   | 고정된 관계, 0은 아닐 수 있음 |
| 전압 크기 | ❌    | 출력은 VCO 레벨         |
| 파형    | ❌    | 새로 생성된 신호          |

👉 **PLL은 신호 복사기가 아님**


## 7️⃣ PLL의 특징

* 피드백 제어 시스템
* Lock 시간 존재
* 지터·위상 잡음 존재
* 분주비 변경으로 주파수 조절 가능

---

## 8️⃣ PLL 종류

* **Integer-N PLL**: 정수 배수
* **Fractional-N PLL**: 분수 배수
* **Analog PLL**
* **Digital PLL (DPLL)**
* **CDR (Clock & Data Recovery)**

---

## 9️⃣ 핵심 키워드 요약

* Lock / Unlock
* Reference Clock
* VCO / KVCO
* Divider (N)
* Loop Bandwidth
* Jitter / Phase Noise

---

## 🔑 최종 한 줄 정리

> **PLL은 기준 신호에 ‘잠겨서’ 출력 클럭의 주파수와 위상을 맞추는 동기화 회로이며,
> 출력은 입력을 복제한 것이 아니라 새로 생성된 신호다.**
