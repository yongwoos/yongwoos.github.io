---
title: 부궤환 증폭기
---
**부궤환 증폭기의 이득 공식:**

$$A_f = \frac{A}{1 + \beta A}$$

- $A_f$ : 궤환 후 이득
- $A$ : 무궤환 이득
- $\beta$ : 궤환율
- $(1 + \beta A)$ : **루프이득(Loop Gain)**

---

### 부궤환 조건 판단

| 조건 | 의미 |
|------|------|
| $(1 + \beta A) > 1$ | **부궤환** → 이득 감소, 안정성 향상 ✅ |
| $(1 + \beta A) = 1$ | 궤환 없음 (무궤환 상태) |
| $(1 + \beta A) < 1$ | **정궤환** → 이득 증가, 발진 가능 |

부궤환이면 $A_f < A$, 즉 이득이 **줄어들어야** 하므로 분모인 $(1+\beta A)$가 **1보다 커야** 합니다.

---

### 부궤환의 장점

- 이득 안정성 향상
- 잡음 및 왜율 감소
- 대역폭 증가
- 입출력 임피던스 개선

## 부궤환(Negative Feedback) 원리
```
입력(Xi) → [+] → [증폭기 A] → 출력(Xo)
            ↑
           [-]
            |
        [궤환회로 β] ←─────────┘
```

출력의 일부를 **입력 측에 되돌려** 원래 입력과 **빼주는(-)** 방식입니다.

---

### 수식으로 이해하기

**① 실제 입력 (오차신호)**
$$X_e = X_i - \beta X_o$$

입력에서 궤환 신호를 **뺍니다.**

**② 출력**
$$X_o = A \cdot X_e = A(X_i - \beta X_o)$$

**③ 정리하면**
$$X_o = \frac{A}{1 + \beta A} \cdot X_i$$

$(1 + \beta A) > 1$ 이므로 → **이득이 줄어듦**

---

### 직관적으로 이해하기

> 🔊 마이크 볼륨을 높이면 스피커 소리가 마이크로 다시 들어가 "삐~" 소리가 나는 것이 **정궤환**
>
> 반대로 소리가 커지면 자동으로 볼륨을 줄여주는 것이 **부궤환**

---

### 부궤환 적용 시 변화

| 특성 | 변화 | 이유 |
|------|------|------|
| 전압이득 | 감소 $(÷(1+\beta A))$ | 분모가 커짐 |
| 대역폭 | 증가 $(×(1+\beta A))$ | 이득×대역폭 = 일정 |
| 왜율(비선형 왜곡) | 감소 $(÷(1+\beta A))$ | 오차 보정 |
| 잡음 | 감소 | 오차 보정 |
| 이득 안정성 | 향상 | A 변해도 $A_f$ 거의 일정 |

---

### 이득 안정성이 좋아지는 이유

$\beta A \gg 1$ 이면:

$$A_f = \frac{A}{1+\beta A} \approx \frac{1}{\beta}$$

→ 증폭기 이득 $A$가 변해도 **$\beta$만으로 이득이 결정**되므로 매우 안정적입니다.

## 이득 A (무궤환 이득)

**증폭기 자체의 고유 증폭 능력**입니다.

$$A = \frac{X_o}{X_i} = \frac{\text{출력}}{\text{입력}}$$

궤환 회로가 **없을 때** 입력 대비 출력이 얼마나 커지는지를 나타냅니다.

---

### 쉽게 비유하면

```
마이크 → [앰프 A=100] → 스피커
```

입력 신호가 1이면 → 출력은 100

> A = 100 이란, 궤환 없이 순수하게 **100배 증폭**한다는 의미

---

### A의 종류 (무엇을 증폭하느냐에 따라)

| 종류 | 정의 | 단위 |
|------|------|------|
| **전압이득** $A_v$ | $V_o / V_i$ | 무차원(배) |
| **전류이득** $A_i$ | $I_o / I_i$ | 무차원(배) |
| **전력이득** $A_p$ | $P_o / P_i$ | 무차원(배) |
| **dB 이득** | $20\log A_v$ | dB |

---

### 궤환 전후 비교

```
궤환 없을 때 → 이득 A (크지만 불안정)
궤환 있을 때 → 이득 Af = A/(1+βA) (작지만 안정)
```

즉 A는 **트랜지스터나 OP-AMP 소자 자체가 가진 증폭 능력**이고, 부궤환을 걸면 그 이득을 **의도적으로 줄여서** 안정성을 얻는 것입니다.

## 궤환율 β (Beta)
**출력 신호 중 얼마만큼을 입력으로 되돌려 보내는 비율**입니다.

$$\beta = \frac{X_f}{X_o} = \frac{\text{궤환 신호}}{\text{출력 신호}}$$

---

### 쉽게 비유하면

```
출력(스피커 소리) → [궤환회로 β] → 입력으로 되돌림
```

> 출력이 100이고 β = 0.1 이면
> → 10만큼 입력으로 되돌려 보냄

---

### β는 무엇으로 만드나?

주로 **저항 분배 회로**로 구현합니다.

```
출력 ──┬── 부하
       │
      [R1]
       │
       ├──→ 궤환 신호 (입력으로)
      [R2]
       │
      GND
```

$$\beta = \frac{R_2}{R_1 + R_2}$$

R1, R2 값으로 **얼마나 되돌릴지** 결정합니다.

---

### β 값의 범위와 의미

| β 값 | 의미 |
|------|------|
| β = 0 | 궤환 없음 (무궤환) |
| 0 < β < 1 | 일반적인 부궤환 |
| β = 1 | 출력 전부를 되돌림 (전압 폴로워) |

---

### A와 β의 관계 정리

```
A  → 증폭기 자체 능력 (크고 불안정)
β  → 되돌려 보내는 비율 (설계자가 결정)
βA → 루프이득 (클수록 안정)

Af = A / (1+βA) ≈ 1/β   (βA>>1 일때)
```

> 결국 β를 설계자가 **저항값으로 정밀하게 설정**하면
> 증폭기 이득이 A가 변해도
> **최종 이득은 1/β로 안정**되는 것입니다 ✅
