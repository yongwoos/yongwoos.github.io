## 슈미트 트리거(Schmitt Trigger) 회로란?

슈미트 트리거 회로는 입력 신호에 포함된 **잡음을 제거**하고 출력 신호를 명확한 **구형파(펄스)**로 변환하기 위해 사용되는 특수한 형태의 비교기입니다.

### 핵심 특징

- **히스테리시스(Hysteresis) 특성**: 일반적인 비교기는 하나의 기준 전압만 가지지만, 슈미트 트리거는 상한 임계값(UTP)과 하한 임계값(LTP)이라는 두 개의 서로 다른 기준 전압을 가집니다. 이 두 임계값의 차이를 히스테리시스라고 합니다.
- **잡음 제거**: 입력 신호에 미세한 잡음이 섞여 있어도 신호가 한 번 상한 임계값을 넘으면 하한 임계값 아래로 떨어지기 전까지는 출력 상태가 변하지 않습니다. 덕분에 출력이 불안정하게 요동치는 현상을 방지할 수 있습니다.
- **정궤환(Positive Feedback)**: 연산증폭기의 비반전(%% + %%) 단자로 출력을 되먹임하여 급격한 상태 변화를 유도하는 구조를 가집니다.

### 동작 방식 (반전형 기준)

1. **입력 전압(%% V_s %%)이 양의 값으로 증가할 때**: 입력이 상한 임계값(UTP)을 넘는 순간 출력(%% V_{out} %%)은 음의 포화 전압으로 급격히 바뀝니다.
2. **입력 전압(%% V_s %%)이 음의 값으로 감소할 때**: 입력이 하한 임계값(LTP)보다 낮아지는 순간 출력(%% V_{out} %%)은 다시 양의 포화 전압으로 급격히 바뀝니다.

### 반전형
반전형 슈미트 트리거는 입력이 상한 임계값을 넘으면 출력이 음의 포화 전압으로 바뀌고, 입력이 하한 임계값보다 낮아지면 출력이 양의 포화 전압으로 바뀌는 구조입니다. 이로 인해 입력 신호가 상하한 임계값 사이에서 요동치더라도 출력은 안정적으로 유지됩니다.


### 비반전형
비반전형 슈미트 트리거는 입력이 상한 임계값을 넘으면 출력이 양의 포화 전압으로 바뀌고, 입력이 하한 임계값보다 낮아지면 출력이 음의 포화 전압으로 바뀌는 구조입니다. 이로 인해 입력 신호가 상하한 임계값 사이에서 요동치더라도 출력은 안정적으로 유지됩니다.
