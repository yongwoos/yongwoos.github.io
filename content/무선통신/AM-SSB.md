반송파제거, 한쪽 측파대만 보내는 방식

반송파와 한쪽 측파대를 제거한 단측파대(SSB) 방식은 양측파대(DSB) 방식보다 대역폭이 절반으로 줄어들고, 송신 전력도 절반으로 줄어듭니다.

동기검파를 사용하여 수신된 SSB 신호를 복조할 때, 송신된 신호의 주파수와 동일한 주파수의 로컬 발진기를 사용하여 원래의 음성 신호를 재생할 수 있습니다. 

SSB 방식은 대역폭과 전력 효율성 측면에서 우수하지만, 복조 과정이 더 복잡하고 정확한 주파수 동기화가 필요하다는 단점이 있습니다. 그러나 이러한 단점에도 불구하고, SSB는 장거리 통신과 음성 통신에서 여전히 널리 사용되고 있습니다.

### SSB vs DSB 기본 비교

```
DSB (Double Sideband)
├── 상측파대 (USB)
├── 반송파
└── 하측파대 (LSB)

SSB (Single Sideband)
└── 상측파대 또는 하측파대 (하나만!)
```

---

### SSB수신기만의 특징 (DSB와 다른 점)

**① 주파수/위상 오차에 더 취약**
```
DSB → 반송파 포함 → 동기 맞추기 쉬움
SSB → 반송파 억압 → 동기 맞추기 어려움
      → 주파수/위상 오차에 민감
```

**② 스피치 클라리파이어 (Speech Clarifier)**
```
SSB 특유의 회로
미세한 주파수 조정 장치
→ 음성을 명료하게 복원하기 위해 필요
```

**④ 좁은 대역폭 필터 사용**
```
DSB 대역폭 = 2W (W: 음성 최고주파수)
SSB 대역폭 = W  (절반!)
→ 더 좁은 필터 필요
```

---

### SSB의 장단점

| 구분 | 내용 |
|------|------|
| **장점** | 대역폭 1/2, 전력효율 높음, 혼신 적음 |
| **단점** | 회로 복잡, 위상오차에 민감, 클라리파이어 필요 |

---

### 핵심 암기

> **국부발진기는 SSB, DSB 모두 사용**
> → SSB만의 특징이 아님 ✅
>
> SSB만의 특징
> = **스피치 클라리파이어** + **좁은 필터** + **위상오차에 민감**
