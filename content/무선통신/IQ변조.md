---
title: IQ 변조
---
> **In-phase(I)와 Quadrature(Q) 두 개의 직교 성분을 이용해 하나의 반송파(carrier)를 변조하는 방식**

IQ 변조: 서로 직교하는 똑같은 주파수의 반송파 cos, sin을 합쳐서 보내는 방식

OFDM: 서로 직교하는 IQ 신호 여러개를 합쳐서 보내는 방식

* 반송파: $\cos(\omega t)$
* 직교 반송파: $\sin(\omega t)$ (90도 위상차)

여기에 두 개의 신호를 따로 실는다:

* I 신호 → cos에 곱함
* Q 신호 → sin에 곱함

그래서 최종 신호는:

$s(t) = I(t)\cos(\omega t) + Q(t)\sin(\omega t)
$

이 방식의 목적은 **한 개의 주파수 대역에 두 개의 독립적인 정보 채널을 실기 위해서**다.

* I = 실수축 (in-phase)
* Q = 허수축 (quadrature)

→ 결국 **복소수 신호 ( I + jQ )** 를 보내는 것과 동일

즉:

* 단순 진폭만 바꾸는 AM보다 정보량 ↑
* 위상 + 진폭 동시에 제어 가능

IQ 평면(복소평면)에서 보면:

* 점 하나 = (I, Q)
* 이 점의 위치가 신호 상태를 의미

예:

* QPSK → 점 4개
* 16QAM → 점 16개

→ 점이 많을수록 데이터 전송량 증가 (대신 노이즈에 취약)

현대 통신 거의 전부:

* LTE / 5G
* Wi-Fi
* 위성통신
* SDR (Software Defined Radio)

IQ 변조를 이해하면 바로 이어지는 개념들:

* QAM (Quadrature Amplitude Modulation)
* OFDM (주파수 다중화)
* 복소 baseband 신호 처리
* RF → baseband 변환 (다운컨버전)

IQ 변조 = **cos/sin 두 축을 이용해서 복소수 형태로 신호를 실어 보내는 고효율 변조 방식**

### 실제 활용
IQ 평면 = 복소평면

한 점 = 하나의 심볼

예를 들어 **16-QAM**:

* I축: {-3, -1, +1, +3}
* Q축: {-3, -1, +1, +3}

→ 가능한 조합 = 16개
→ 한 심볼 = 4비트 (2⁴)

즉:

* 비트 → (I, Q) 좌표로 매핑
* 그 좌표가 IQ 변조를 통해 RF로 실림

직관:

* 점이 멀리 떨어질수록 노이즈 강함
* 점이 많을수록 데이터량 ↑ 대신 오류 ↑

그래서 실제 시스템은:

* 채널 좋으면 64QAM, 256QAM
* 나쁘면 QPSK로 내려감 (adaptive modulation)

#### OFDM에서 사용
각 서브캐리어마다:

* 하나의 QAM 심볼이 실림
* 즉, 각 서브캐리어 = 하나의 IQ 점

구조:

1. 데이터 비트 입력
2. QAM 매핑 → 복소수 심볼들 생성
3. 이걸 주파수 축에 배치
4. IFFT 수행 → 시간 신호 생성
5. IQ 신호로 변환 → RF 송신

핵심 포인트:

* OFDM은 “여러 개의 IQ 신호를 한 번에 보내는 구조”

#### IQ 송신기 구조

디지털 → RF 흐름:

1. 비트 스트림
2. QAM 매핑 (→ I/Q 심볼)
3. IFFT (OFDM이면)
4. DAC (디지털 → 아날로그)
5. IQ Mixer

여기서 핵심:

* I 신호 → cos(ωt)
* Q 신호 → sin(ωt)

두 개 합쳐서 RF 신호 생성

#### 수신기 구조
```
1. RF 수신
2. LNA (증폭)
3. Mixer (다운컨버전)
4. ADC
5. FFT
6. QAM 디코딩
```
