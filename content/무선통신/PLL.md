### PLL(Phase Locked Loop)이란?

**위상 동기 루프**

입력 신호의 **주파수와 위상**을 추적하여
출력 신호를 **자동으로 동기**시키는 회로

---

### 기본 구성 (문제 23번)

```
입력신호 →[위상검출기(PD)]→[저역통과필터(LPF)]→[전압제어발진기(VCO)]→ 출력
              ↑                                           |
              └───────────────────────────────────────────┘
                              궤환(Feedback)
```

---

### 각 블록의 역할

**① 위상검출기 (Phase Detector)**
```
입력신호와 VCO 출력신호의
위상차를 비교
→ 위상차에 비례한 오차전압 출력
```

**② 저역통과필터 (LPF)**
```
위상검출기 출력의
고주파 잡음 제거
→ 깨끗한 직류 오차전압만 통과
```

**③ 전압제어발진기 (VCO)**
```
Voltage Controlled Oscillator
입력 전압에 따라
발진 주파수가 변함

전압 높음 → 주파수 증가
전압 낮음 → 주파수 감소
```

---

### 동작 원리 (Lock 과정)

**① 초기 상태 (unlock)**
```
입력신호 주파수 : 100MHz
VCO 출력 주파수 : 90MHz
→ 주파수/위상 차이 존재
```

**② 오차 검출**
```
위상검출기가
두 신호의 위상차 감지
→ 오차전압 발생
```

**③ VCO 제어**
```
오차전압 → LPF → VCO 입력
VCO 주파수가 점점 변함
90MHz → 95MHz → 99MHz → 100MHz
```

**④ Lock 상태**
```
입력신호 : 100MHz
VCO 출력 : 100MHz
→ 위상차 = 0
→ 오차전압 = 0
→ 완전 동기! ✅
```

---

### 부궤환과의 관계

```
PLL도 부궤환 시스템!

오차가 크면 → 많이 보정
오차가 작면 → 조금 보정
오차가 0이면 → 보정 안함 (Lock!)
```

---

### PLL 응용 분야

| 응용 | 설명 |
|------|------|
| **FM 복조** | 주파수 변화 추적→음성 복원 |
| **주파수 합성** | 정밀한 주파수 생성 |
| **클럭 동기** | 디지털 회로 동기화 |
| **모터 속도제어** | 일정 속도 유지 |
| **위성통신** | 반송파 동기 |

---

### 핵심 암기

> **PD → LPF → VCO → 궤환**
>
> 위상검출 → 잡음제거 → 주파수조정 → 다시비교
>
> 이 과정을 반복해서
> 결국 입력과 출력이 **완전히 동기(Lock)** 되는 것! ✅
