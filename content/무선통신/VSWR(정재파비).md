---
title: VSWR(정재파비)
---
송신기와 안테나 사이의 임피던스가 얼마나 잘 일치(정합)하는지를 나타내는 지표

정합이 완벽할수록 반사되는 에너지가 적어 효율적인 송수신이 가능합니다.

즉, 1에 가까울 수록 반사손실이 적은(양호한) 것

$$
VWSR = \frac{V_{max}}{V_{min}} = \frac{1 + |\Gamma|}{1 - |\Gamma|}
$$

VSWR의 범위는 $1 \le VSWR \le \infty$

### 반사계수
$\Gamma$는 반사 계수로, 안테나에서 반사되는 전력의 비율

$$
\Gamma = \frac{Z_L - Z_0}{Z_L + Z_0}
$$
반사계수의 범위는 $0 \le |\Gamma| \le 1$

1에 가까울 수록 반사가 많이 되는 것

### 반파장 안테나 전류분포
중앙 전류 최대, 양끝 전류 0, 위상은 모든 점에서 동일


```
안테나 끝 → 개방단
→ 전류 반사 발생
→ 진행파 + 반사파 = 정재파 형성
→ 전류 분포 생김
```

---

### 진행파 + 반사파 과정

```
급전점에서 전류 공급
      ↓
전류가 안테나 위아래로 진행
      ↓
안테나 끝(개방단)에서 반사
      ↓
진행파 + 반사파 합성
      ↓
정재파 형성!
```

---

### 결과적으로

```
진행파  : sin(ωt - βx)  →→→
반사파  : sin(ωt + βx)  ←←←
합성    : cos(βx) × sin(ωt)
           ↑           ↑
        위치마다      위상은
        진폭 다름     동일!
```

**안테나 전류 분포:**
```
끝   : cos(β×λ/4) = 0   → 전류 최소
중앙 : cos(β×0)   = 1   → 전류 최대
끝   : cos(β×λ/4) = 0   → 전류 최소

위상항 sin(ωt)는 변하지 않음!
→ 모든 점에서 위상 동일 ✅
```

---

### 핵심 정리

> **안테나 끝 = 개방단 = 전류 반사**
>
> 반사파 때문에 정재파 형성
> → 위치마다 **진폭(크기)** 달라짐
> → **위상**은 모든 점에서 동일
>
> 전류 크기 차이 = 반사파 때문 ✅
> 위상 차이 = 없음 ✅
