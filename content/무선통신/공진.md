### 공진이란?

**LC 회로에서 유도성 리액턴스와
용량성 리액턴스가 같아지는 현상**

$$X_L = X_C$$

---

### L과 C의 특성 비교

**인덕터 (L):**
```
주파수 높을수록 → 저항(리액턴스) 커짐
전류 변화를 방해
에너지를 자기장으로 저장
```
$$X_L = 2\pi f L$$

**커패시터 (C):**
```
주파수 높을수록 → 저항(리액턴스) 작아짐
전압 변화를 방해
에너지를 전기장으로 저장
```
$$X_C = \frac{1}{2\pi f C}$$

---

### 주파수에 따른 변화

```
주파수 ↑        XL ↑        XC ↓

낮은 주파수 → XC > XL (용량성)
공진 주파수 → XC = XL (공진!)
높은 주파수 → XL > XC (유도성)
```

그래프로 보면:
```
리액턴스
↑    XL /
     /
     /  ← 공진점
    /×
   / \  XC
  /   \
──────────→ 주파수
      f0
```

---

### 공진의 핵심 원리

**에너지 교환:**
```
① 커패시터 충전
   전기장에 에너지 저장
        ↓
② 커패시터 방전
   인덕터로 에너지 전달
        ↓
③ 인덕터 자기장에 에너지 저장
        ↓
④ 인덕터 에너지 방출
   커패시터 충전
        ↓
① 반복!
```

**쉽게 비유:**
```
그네 타기와 같음!

그네를 밀면 (에너지 입력)
앞뒤로 왔다갔다 (L↔C 에너지 교환)
고유 주기로 진동 (공진주파수)
```

---

### 직렬 공진 vs 병렬 공진

**직렬 공진:**
```
L과 C가 직렬 연결

    L       C
───UUUU───||───
        ↑
     공진 시
   임피던스 최소
   전류 최대 ✅
```

| 조건 | 임피던스 | 전류 |
|------|------|------|
| f < f0 | 큼 | 작음 |
| f = f0 | **최소** | **최대** |
| f > f0 | 큼 | 작음 |

**병렬 공진:**
```
L과 C가 병렬 연결

───┬─UUUU─┬───
   └──||──┘
        ↑
     공진 시
   임피던스 최대
   전류 최소 ✅
```

| 조건 | 임피던스 | 전류 |
|------|------|------|
| f < f0 | 작음 | 큼 |
| f = f0 | **최대** | **최소** |
| f > f0 | 작음 | 큼 |

---

### Q값 (Quality Factor)

**공진의 날카로움을 나타내는 지표**

$$Q = \frac{f_0}{BW} = \frac{X_L}{R}$$

```
Q 높음 → 공진이 날카로움
         특정 주파수만 통과
         선택도 높음 ✅

Q 낮음 → 공진이 완만함
         넓은 주파수 통과
         선택도 낮음
```

그래프:
```
출력
↑    Q높음     Q낮음
|     /\       /──\
|    /  \    /     \
|   /    \  /       \
└──────────────────→ f
        f0
```

---

### 공진 주파수 공식

$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

**L, C와의 관계:**

| 변화 | 공진주파수 |
|------|------|
| L 증가 | f0 감소 |
| L 감소 | f0 증가 |
| C 증가 | f0 감소 |
| C 감소 | f0 증가 |

---

### 실생활 응용

```
라디오 채널 선택
→ 가변 커패시터로 C값 조절
→ 공진주파수 변경
→ 원하는 방송국 선택! 📻

안테나 공진
→ 특정 주파수에서 공진
→ 해당 주파수 신호 최대 수신 📡
```

---

### 핵심 암기

> **공진 = XL = XC**
>
> 직렬공진 → 임피던스 최소, 전류 최대
> 병렬공진 → 임피던스 최대, 전류 최소
>
> $$f_0 = \frac{1}{2\pi\sqrt{LC}}$$ ✅

## 공진 시 위상 관계
> **공진 시 전압과 전류의 위상이 같아집니다 (동위상)**

---

### 왜 그런가?

**각 소자별 위상 특성:**

```
저항 R    → 전압과 전류 위상 동일 (0°)
인덕터 L  → 전압이 전류보다 90° 앞섬
커패시터C → 전류가 전압보다 90° 앞섬
```

**쉽게 암기:**
```
L : ELI  → E(전압)가 I(전류)보다 앞섬
C : ICE  → I(전류)가 E(전압)보다 앞섬
```

---

### 공진 시 위상이 같아지는 이유

```
XL이 만드는 위상 = +90°
XC이 만드는 위상 = -90°

공진 시 XL = XC 이므로

(+90°) + (-90°) = 0°

→ 위상차 = 0° → 동위상! ✅
```

---

### 직렬 공진 위상 변화

**f < f0 (공진 이하):**
```
XC > XL
용량성 회로
전류가 전압보다 앞섬 (-)
```

**f = f0 (공진):**
```
XC = XL
저항성 회로
전압 = 전류 위상 동일 ✅
```

**f > f0 (공진 이상):**
```
XL > XC
유도성 회로
전압이 전류보다 앞섬 (+)
```

그래프로 보면:
```
위상차
+90°|          /
    |         /  유도성
  0°|────────×──────→ f
    |       / f0
-90°|      /
    |용량성
```

---

### 임피던스 관점에서 보면

$$Z = R + j(X_L - X_C)$$

**공진 시:**
$$X_L = X_C$$
$$Z = R + j(0) = R$$

```
허수부 = 0
→ 순수 저항만 남음
→ 저항은 위상차 없음
→ 전압 전류 동위상! ✅
```

---

### 핵심 암기

> **공진 전 → 위상차 있음**
> **공진 시 → 위상차 = 0° (동위상)**
> **공진 후 → 위상차 있음**
>
> 공진 = XL과 XC가 서로 상쇄
> → 마치 저항만 있는 회로처럼 동작! ✅
