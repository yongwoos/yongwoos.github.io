---
title: 정합필터
---
정합필터(Matched Filter)는 **잡음 환경에서 신호 대 잡음비(SNR)를 최대화**하기 위해 설계된 최적 선형 필터입니다.

## 기본 원리

수신 신호가 다음과 같을 때:

> **r(t) = s(t) + n(t)**
> (s(t): 원하는 신호, n(t): 백색 가우시안 잡음)

정합필터의 임펄스 응답은 **탐지하려는 신호를 시간 반전한 형태**입니다:

> **h(t) = s(T − t)**

즉, 필터의 주파수 응답은 신호 스펙트럼의 켤레 복소수에 비례합니다:

> **H(f) = S*(f) · e^(−j2πfT)**

## 핵심 특성

| 특성 | 설명 |
|------|------|
| 최적성 | 백색 잡음 조건에서 SNR을 수학적으로 최대화 |
| 출력 SNR | `2E/N₀` (E: 신호 에너지, N₀: 잡음 전력 스펙트럼 밀도) |
| 출력 파형 | 신호의 자기상관함수(autocorrelation) |
| 판정 시각 | t = T 에서 SNR 최대 |

## 동작 방식

```
수신신호 r(t) → [정합필터 h(t)=s(T-t)] → 출력 y(t) → t=T에서 샘플링 → 판정
```

출력 y(T)는 수신신호와 기준신호의 **상관(correlation)**을 계산한 것과 동일합니다:

$$y(T) = \int_{-\infty}^{\infty} r(\tau) \cdot s(\tau) \, d\tau$$

따라서 정합필터는 **상관수신기(Correlator)**와 수학적으로 동치입니다.

## 주요 응용 분야

- **레이더 / 소나** — 표적 탐지 및 거리 측정
- **디지털 통신** — 심볼 판정 (BPSK, QPSK 등)
- **GPS** — 위성 신호 포착 및 추적
- **MRI** — 생체 신호 처리
- **지진파 탐지** — 약한 신호 검출

## 설계 조건

최적 SNR을 얻으려면:
1. 필터가 탐지 대상 신호에 **정확히 매칭**되어야 함
2. 잡음이 **백색(white)** 이어야 함 (유색 잡음이면 백색화 전처리 필요)
3. 신호 파형을 **사전에 알고 있어야** 함 (수동 탐지에는 부적합)

## 정합필터의 전제 조건

정합필터는 **탐지하려는 신호 s(t)의 파형을 설계자가 미리 알고 있어야** 합니다.

> h(t) = s(T − t)

필터 자체가 신호의 시간반전 형태이기 때문에, 신호를 모르면 필터를 만들 수 없습니다.

---

## 그래서 제가 앞서 쓴 표현이 애매했는데...

제가 마지막에 쓴

> "신호 파형을 사전에 알고 있어야 함 **(수동 탐지에는 부적합)**"

이 말은 **탐지 대상이 어떤 신호를 쓸지 모르는 경우** (예: 적 레이더 신호를 몰래 감청하는 수동 탐지)에는 적용하기 어렵다는 뜻이었습니다.

## 정리하면

| 상황 | 정합필터 적용 가능? |
|------|----------------|
| 내가 송신한 신호를 내가 수신 (레이더, 통신) | ✅ 가능 (신호 파형 알고 있음) |
| 상대방 신호를 수신 (수동 탐지) | ❌ 어려움 (파형 모름) |

레이더나 디지털 통신은 **자기가 보낸 신호를 알고 있기 때문에** 정합필터가 잘 작동하는 대표적인 예입니다.


## 사각형파 정합필터 예시

사각형파 s(t)에 대해 정합필터 h(t) = s(T−t)도 **사각형파** (시간반전해도 동일한 모양)

### 컨볼루션 결과

```
s(t)        h(t)=s(T-t)      y(t) = s(t) * h(t)

 ___          ___                 /\
|   |    *   |   |    =          /  \
|   |        |   |              /    \
```

사각형 × 사각형 컨볼루션 → **삼각형** 출력

---

## 샘플링 시점

삼각형의 **꼭짓점 (t = T)** 에서 샘플링 → 이 순간이 **SNR 최대**

```
y(t)
  ▲
  |     /*\  ← 여기서 샘플링 (t = T)
  |    /   \
  |   /     \
  |  /       \
  | /         \
  +------------→ t
  0     T     2T
```

---

## 직관적 이해

꼭짓점에서 샘플링하는 이유는:
- 신호 성분이 **완전히 누적**된 시점
- 신호 에너지를 **최대한 모은** 순간
- 수식으로는 신호와 필터의 **완전한 overlap** 상태

그래서 정합필터 = "신호를 끝까지 다 모은 다음 판단한다"는 개념입니다.
