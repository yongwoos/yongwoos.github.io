---
title: 전파투시도
---

```
전파투시도에서 전파 통로는
직선으로 나타냅니다
```

실제로는 대기 굴절로 전파가 휘어지지만
전파투시도에서는 **등가지구 반경을 사용**하여
전파 경로를 **직선으로 단순화**합니다

---

### 전파투시도란?

**송수신 지점 사이의 지형 단면도**

```
송신점                        수신점
  ×─────────────────────────×
  |        전파 경로(직선)        |
  |    ↗산악            ↘      |
  |  ↗                   ↘    |
  |↗_____________________↘   |
         지형 단면
```

전파가 장애물을 통과할 수 있는지
**시각적으로 확인**하는 도면

---

### 전파투시도의 목적

```
① 전파 경로상 장애물 파악
② 회절손실 계산
③ 통신 가능 여부 판단
④ 안테나 높이 결정
```

---

### 등가지구 반경계수 (k)

**실제 지구 반경을 보정하는 계수**

```
실제 전파는 대기 굴절로
약간 휘어져 진행

이를 보정하기 위해
지구 반경을 크게 가정
→ 전파 경로를 직선으로 표현 가능!
```

$$R_{eq} = k \times R_{re}$$

```
표준 대기 조건 : k = 4/3
→ 지구 반경을 4/3배로 키워서
→ 전파 경로를 직선으로 표현
```

| ② 대지와 수직인 지형 단면도 | 옳음 ✅ | 전파투시도의 정의 |
| ③ 등가지구 반경계수 고려 | 옳음 ✅ | k값 사용 |
| ④ 수직 장애물 효과 연구 | 옳음 ✅ | 회절 분석에 활용 |

### 핵심 암기

>
> 전파 경로 → **직선** (곡선 ❌)
> 등가지구 반경계수(k) 사용
> 장애물의 영향 분석에 활용 ✅
