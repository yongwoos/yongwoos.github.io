### FM 신호의 수식 표현

$$s(t) = A_c \cos\left(2\pi f_c t + 2\pi k_f \int_0^t m(\tau)d\tau\right)$$

순시 주파수: $f_i(t) = f_c + k_f \cdot m(t)$

주파수 편이: $\Delta f = k_f \cdot A_m$


#### 1. 경사 검파기 (Slope Detector)
가장 단순한 방식. LC 회로의 공진 곡선 경사면을 이용해 주파수 변화를 진폭 변화로 변환 후 AM 검파.

* **단점**: 선형 범위가 좁고 왜곡이 크며 진폭 변동에 민감 → 실용성 낮음

#### 2. Foster-Seeley 판별기

두 개의 다이오드와 중심 주파수에 동조된 이중 회로를 사용합니다.

$$V_{out} = V_{D1} - V_{D2}$$

* 중심 주파수: $V_{D1} = V_{D2}$ → 출력 0
* 주파수 상승: $V_{D1} > V_{D2}$ → 출력 양(+)
* 주파수 하강: $V_{D1} < V_{D2}$ → 출력 음(−)

**특징**
* 선형성이 양호
* **진폭 변동에 민감** → 앞단에 리미터(Limiter) 필수

---

#### 3. 비율 검파기 (Ratio Detector)

Foster-Seeley를 개선한 방식. 다이오드 방향을 하나 반전시키고 대용량 커패시터 추가.

$$V_{out} = \frac{V_{D1} - V_{D2}}{2}$$

**특징**
* **진폭 변동 자체 억압** (리미터 불필요)
* 회로가 단순하고 경제적
* AM 라디오 IC에 많이 내장됨

---

#### 4. 위상 고정 루프 (PLL, Phase-Locked Loop)

$$\text{VCO 제어 전압} = V_{out} \propto \Delta f$$

PLL이 입력 FM 신호의 주파수를 추적하면, VCO 제어 전압이 곧 복조 신호가 됩니다.

```
FM 입력 → 위상 검출기 → LPF → [복조 출력]
               ↑                      ↓
              VCO ←────────────────────
```

**특징**
* 선형성 우수
* IC 구현에 적합 (LM565, NE564 등)
* 포착 범위(Capture Range) / 유지 범위(Lock Range) 설계 중요

---

#### 5. 직교 검파기 (Quadrature Detector)

가장 널리 쓰이는 IC 방식 (예: MC13600, LA1800 등)

입력 신호를 90° 위상 지연시킨 신호와 곱셈(Multiplier)합니다.

$$V_{out} = \sin(\Delta\phi) \approx \Delta\phi \propto \Delta f$$

```
IF 입력 ──┬──────────────→ [Multiplier] → LPF → 출력
          └→ [90° 지연회로] ↗
```

**특징**
* 소수의 외부 소자로 구현 가능 (동조 코일 1개)
* FM 수신기 IC의 사실상 표준
* 선형성 양호

---

### 방식 비교

| 방식 | 선형성 | 리미터 필요 | IC 집적 | 현재 사용 |
|------|--------|------------|--------|--------|
| 경사 검파 | 나쁨 | 필요 | 불가 | 거의 없음 |
| Foster-Seeley | 양호 | **필요** | 어려움 | 드묾 |
| 비율 검파 | 양호 | 불필요 | 부분 | 일부 |
| PLL | 우수 | 불필요 | 적합 | 많음 |
| 직교 검파 | 우수 | 불필요 | **최적** | **표준** |

---

### 복조 후 처리

검파 출력에는 **디엠파시스(De-emphasis)** 회로가 따라옵니다.

* FM 송신 시 고음역을 강조(Pre-emphasis, 75 μs)하여 S/N비 개선
* 수신 후 반대로 고음역을 감쇠(De-emphasis)하여 원래 특성 복원

$$\tau = RC = 75\ \mu s \quad \text{(한국/미국/일본 기준)}$$

## 포스터 실리 vs 비검파
## 포스터 실리 & 비검파 원리

---

### FM 검파란?

FM 신호에서 **주파수 변화 → 전압 변화**로 바꾸는 것

```
FM 입력 → 주파수 변화 감지 → 음성/데이터 출력
```

---

## 1. 포스터 실리 검파 (Foster-Seeley)

### 회로 구성
```
FM입력 → [리미터] → [1차코일] → [2차코일(센터탭)] → [다이오드D1,D2] → 출력
```

### 동작 원리

**중심 주파수(fc)일 때:**
```
1차코일과 2차코일 → 90° 위상차
D1, D2에 걸리는 전압 → 동일
출력 = V1 - V2 = 0V
```

**fc보다 높을 때:**
```
위상차 90° 이상
D1 전압 > D2 전압
출력 = 양(+)전압
```

**fc보다 낮을 때:**
```
위상차 90° 이하
D1 전압 < D2 전압
출력 = 음(-)전압
```

### 핵심 포인트
```
주파수 변화 → 위상차 변화 → 전압차 변화 → 출력
```

- **위상 변화**를 이용해 FM 복조
- 진폭 변화에 민감 → **리미터 필수**

---

## 2. 비검파 (Ratio Detector)

### 회로 구성
```
FM입력 → [1차코일] → [2차코일] → [다이오드D1,D2] → 출력
                                         ↓
                                    [대용량 커패시터C]
                                    (진폭 안정화)
```

### 포스터 실리와의 차이점

**다이오드 방향이 다름:**
```
포스터 실리 → D1, D2 같은 방향
비검파     → D1, D2 반대 방향
```

### 동작 원리

**대용량 커패시터의 역할:**
```
진폭이 갑자기 변해도
커패시터가 전압을 일정하게 유지
→ 리미터 없이도 진폭 변화 억제!
```

**출력 전압:**
```
포스터실리 → V1 - V2 (차)
비검파     → V1 / V2 (비율) ← 이름의 유래!
```

---

## 두 회로 최종 비교

| 구분 | 포스터 실리 | 비검파 |
|------|------|------|
| 원리 | 위상차 → 전압차 | 전압 비율 검출 |
| 리미터 | 필요 | 불필요 |
| 진폭억제 | 리미터로 해결 | 대용량 C로 해결 |
| 검파감도 | 높음 (2) | 낮음 (1) |
| 출력 | V1 - V2 | V1/V2 비율 |
| 장점 | 감도 높음 | 회로 간단 |
| 단점 | 리미터 필요 | 감도 낮음 |

---

### 쉽게 암기하는 법

> **포스터 실리**
> = 감도는 좋은데 예민해서 (진폭변화에)
> = 리미터로 보호가 필요한 섬세한 회로 🎯
>
> **비검파**
> = 둔감하지만 (감도 낮음)
> = 스스로 진폭을 조절하는 강인한 회로 💪
