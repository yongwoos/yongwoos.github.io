## IDC란?

**Instantaneous Deviation Control (순시 편이 제어)**

FM 송신 시 주파수 편이가 **규정된 최대값을 넘지 않도록** 자동으로 제어하는 회로

### 왜 필요한가?

```
FM 변조 원리:
입력 음성 진폭이 클수록 → 주파수 편이 커짐

문제:
음성이 너무 크면 → 편이가 규정치 초과
→ 인접 채널 간섭 발생! ❌
```

**예시:**
```
FM 방송 최대 주파수편이 = ±75kHz (규정)

입력이 너무 크면
→ ±100kHz까지 편이 발생
→ 옆 채널 침범! ❌

IDC 동작
→ 자동으로 ±75kHz 이내로 제한 ✅
```

### FM 송신기 전체 구성

```
마이크 → [프리엠파시스] → [IDC] → [FM변조기] → [전력증폭] → 안테나
              ↑                ↑
         고음역 강조      편이 제한
```


### 나머지 선택지 정리

| 선택지 | 실제 담당 회로 |
|------|------|
| ① 변조신호의 진폭 강화 | 프리엠파시스 회로 |
| ② 임계현상 억제 | 리미터 회로 |
| **③ 최대 주파수편이 규정치 유지** | **IDC 회로** ✅ |
| ④ 도래전파 없을 때 잡음억제 | 스켈치 회로 |


### 핵심 암기

> **IDC = 주파수 편이 경찰관** 👮
>
> 편이가 규정치(±75kHz)를 넘으려 하면
> 즉시 제어해서 규정 범위 안으로 유지! ✅
