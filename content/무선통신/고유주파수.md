### 핵심 공식

$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

### 고유 주파수란?

```
안테나가 외부 에너지 없이
스스로 진동하는 주파수

LC 공진 회로와 동일한 원리!
```

**공진 시:**
```
유도성 리액턴스 = 용량성 리액턴스
XL = XC
2πfL = 1/(2πfC)
→ 이를 만족하는 f가 고유주파수!
```

---

### LC 공진 개념

```
L (인덕터) ↔ C (커패시터)
에너지를 서로 주고받으며 진동

L 크면 → f 낮아짐
C 크면 → f 낮아짐
L,C 작으면 → f 높아짐
```

---

### 핵심 암기

$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

> **L, C 둘 다 2이면**
> $$f_0 = \frac{1}{2\pi\sqrt{4 \times 10^{-18}}} = 80[MHz]$$ ✅
