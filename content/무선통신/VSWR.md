---
title: VSWR(정재파비)
---
$$
VWSR = \frac{V_{max}}{V_{min}} = \frac{1 + |\Gamma|}{1 - |\Gamma|}
$$

$$
\Gamma = \frac{Z_L - Z_0}{Z_L + Z_0}
$$

여기서 \(\Gamma\)는 반사 계수로, 안테나에서 반사되는 전력의 비율을 나타냅니다. 

VSWR 값이 1에 가까울수록 반사되는 전력이 적고, 시스템이 효율적으로 작동함을 의미합니다. 일반적으로 VSWR 값이 2:1 이하인 경우는 허용 가능한 범위로 간주됩니다.

### 반파장 안테나 전류분포
중앙 전류 최대, 양끝 전류 0, 위상은 모든 점에서 동일


```
안테나 끝 → 개방단
→ 전류 반사 발생
→ 진행파 + 반사파 = 정재파 형성
→ 전류 분포 생김
```

---

### 진행파 + 반사파 과정

```
급전점에서 전류 공급
      ↓
전류가 안테나 위아래로 진행
      ↓
안테나 끝(개방단)에서 반사
      ↓
진행파 + 반사파 합성
      ↓
정재파 형성!
```

---

### 결과적으로

```
진행파  : sin(ωt - βx)  →→→
반사파  : sin(ωt + βx)  ←←←
합성    : cos(βx) × sin(ωt)
           ↑           ↑
        위치마다      위상은
        진폭 다름     동일!
```

**안테나 전류 분포:**
```
끝   : cos(β×λ/4) = 0   → 전류 최소
중앙 : cos(β×0)   = 1   → 전류 최대
끝   : cos(β×λ/4) = 0   → 전류 최소

위상항 sin(ωt)는 변하지 않음!
→ 모든 점에서 위상 동일 ✅
```

---

### 핵심 정리

> **안테나 끝 = 개방단 = 전류 반사**
>
> 반사파 때문에 정재파 형성
> → 위치마다 **진폭(크기)** 달라짐
> → **위상**은 모든 점에서 동일
>
> 전류 크기 차이 = 반사파 때문 ✅
> 위상 차이 = 없음 ✅
