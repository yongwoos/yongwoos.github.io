## 위성통신 중계방식
중계기(Transponder)는 위성에서 신호를 수신하여 주파수를 변환하고 증폭한 뒤 다시 지상으로 송신하는 장치입니다. 위성 중계 방식은 크게 다음과 같이 분류됩니다.

* **직접 중계방식 (Direct / RF-to-RF Conversion):** 수신된 미약한 신호를 주파수만 변환하여 전력 증폭기(TWTA 등)로 증폭한 후 그대로 다시 송신하는 방식입니다. 구조가 비교적 간단하여 **통신 및 방송 위성**에서 널리 사용됩니다.

* **헤테로다인 중계방식 (Heterodyne):** 수신 신호를 중간 주파수(IF)로 변환하여 증폭한 뒤 다시 송신 주파수로 변환하는 방식입니다. 직접 중계방식의 일종으로 볼 수 있습니다.

* **재생 중계방식 (Regenerative / Demod-Remod):** 위성에서 수신 신호를 일단 복조(Demodulation)하여 원래의 디지털 신호로 재생한 후, 다시 변조하여 송신하는 방식입니다. 신호 품질은 좋으나 위성의 구조가 복잡해집니다.

## 마이크로파 중계방식
### ① 검파 중계 방식

```
수신 → [복조] → [증폭] → [재변조] → 송신
         ↑
      신호를 일단 검파(복조)후 다시 변조
```

- 신호를 **베이스밴드까지 복조** 후 재변조
- 잡음이 누적되지 않음
- 회로 복잡

---

### ② 재생 중계 방식

```
수신 → [복조] → [파형정형] → [재변조] → 송신
                    ↑
               디지털 신호 재생성
```

- 디지털 신호를 **완전히 재생성**
- 잡음 누적 **완전 차단** ✅
- 디지털 통신에 최적

```
0101 수신 → 잡음 섞임 → 파형정형 → 깨끗한 0101 재생
```

---

### ③ 무급전 중계 방식

```
송신 → [반사판/렌즈] → 수신
           ↑
       전원 불필요!
```

- **전원 없이** 반사판만으로 중계
- 산악지형 등 전원공급 어려운 곳에 사용
- 구조 단순, 유지보수 용이

---

### 중계 방식 총정리

| 방식 | 원리 | 특징 |
|------|------|------|
| **검파 중계** | 복조→증폭→재변조 | 잡음 누적 적음 |
| **재생 중계** | 디지털 신호 재생 | 잡음 완전 차단 |
| **무급전 중계** | 반사판 이용 | 전원 불필요 |
| ~~반파 중계~~ | 존재하지 않음 | ❌ |

---

### 핵심 암기

> 마이크로웨이브 중계방식
> = **검파** + **재생** + **무급전**
>
> **반파 중계는 없는 방식!** ✅
