---
title: 상호변조/혼변조
---
두 개 이상의 서로 다른 주파수 신호가 비선형 회로(증폭기 등)를 통과할 때, 이들이 서로 혼합되어 원래 신호에는 없던 새로운 주파수 성분(합과 차의 형태)을 만들어내는 현상입니다. 

> 통과대역 밖에 존재하는 강력한 방해파가 통과대역 내의 희망파에 방해를 미쳐 통과대역 밖의 방해파에 의해 통과대역 내의 희망파가 영향을 받게 되는 현상

### 혼변조 (Intermodulation)란?

**2개 이상의 강한 신호가 비선형 소자를 통과할 때
서로 섞여서 새로운 주파수 성분을 만들어내는 현상**

---

### 발생 원리

```
방해파 f1, f2 → [비선형 소자] → 새로운 주파수 생성

새로운 주파수 = mf1 ± nf2
(m, n = 정수)
```

**예시:**
```
f1 = 100MHz
f2 = 101MHz 가 들어오면

2f1 - f2 = 99MHz  ← 희망파 대역 침범!
2f2 - f1 = 102MHz ← 희망파 대역 침범!
```

---

### 이외

**① 스퓨리어스 레스폰스 (Spurious Response)**
```
수신기가 원하지 않는 주파수에
반응하는 현상
(영상주파수 간섭 등)
```

**③ 잡음감도**
```
수신기가 감지할 수 있는
최소 잡음 레벨
```

**④ 감도 억압효과**
```
강한 방해신호가 들어올 때
원하는 신호의 감도가
저하되는 현상
```

---

### 혼변조 vs 감도억압 비교

| 구분 | 혼변조 | 감도억압 |
|------|------|------|
| 원인 | 비선형 소자 | 강한 방해신호 |
| 결과 | 새 주파수 생성 | 희망파 감도 저하 |
| 방해파 수 | 2개 이상 | 1개 이상 |

---

### 혼변조 방지 대책

```
① 선형성 좋은 소자 사용
② 입력단 필터 강화
③ 방해파 차단
④ 수신기 다이나믹 레인지 확대
```

---

### 핵심 암기

> **혼변조 = 주파수가 서로 섞여(혼합)**
> **새로운 방해 주파수를 만들어내는 현상** ✅
>
> 통과대역 **밖**의 신호가
> 통과대역 **안**의 신호를 방해! 👿
