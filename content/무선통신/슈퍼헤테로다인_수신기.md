---
title: 슈퍼헤테로다인 수신기
---
> 수신된 높은 주파수의 신호를 처리가 쉬운 중간 주파수로 변환하여 증폭 및 필터링을 수행하는 것

1. RF 증폭기 - 안테나로 들어온 미세한 신호를 증폭하고 원하는 대역만 1차적으로 걸러냅니다.
2. 국부 발진기 - 수신기가 내부적으로 생성하는 주파수$f_{LO}$. 사용자가 채널을 바꿀 때 이 LO 주파수가 변합니다.
3. 혼합기 (Mixer) - RF 신호($f_{RF}$)와 LO 신호를 섞습니다. 이때 두 주파수의 차이에 해당하는 중간 주파수($f_{IF} = |f_{RF} - f_{LO}|$)가 생성됩니다.
4. IF 증폭기 및 필터 - 신호가 어떤 주파수로 들어왔든 항상 동일한 IF 대역으로 변환되므로, 여기서 아주 정밀하게 신호를 뽑아내고 증폭할 수 있습니다.
5. 검파기 (Detector) - 중간 주파수에서 실제 데이터(음성, 영상 등)를 추출합니다.

```
영상주파수 = 수신주파수 + 중간주파수*2

중간주파수 = 수신주파수 - 국부발진기주파수
```

## 영상 주파수
수학적으로 $|f_{RF} - f_{LO}|$를 계산할 때, 우리가 원하는 신호 외에도 반대편에 위치한 원치 않는 신호가 동일한 IF로 변환되는 현상이 발생합니다. 이를 **이미지 주파수**라고 하며, 이를 제거하기 위해 혼합기 앞단에 성능 좋은 RF 필터가 필요합니다.
