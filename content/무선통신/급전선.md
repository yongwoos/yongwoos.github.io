### 급전선이란?

**송신기/수신기와 안테나를
연결하는 전송선로**

```
송신기 → [급전선] → 안테나 → 전파방사
```

---

### 동조 급전선

```
특징:
정재파가 분포하는 급전선

송신기 ═══════════ 안테나
       정재파 존재
```

**장단점:**
```
장점:
→ 정합장치 불필요
→ 구조 단순

단점:
→ 전력손실 큼
→ 길이 제한 (λ/2 배수)
→ 주파수 변경 어려움
```

**주로 사용:**
```
거리가 짧을 때
단일 주파수 사용 시
```

---

### 비동조 급전선

```
특징:
정재파 없이 진행파만 존재

송신기 ─────────── 안테나
       진행파만 존재
```

**장단점:**
```
장점:
→ 전력손실 적음
→ 길이 제한 없음
→ 주파수 변경 용이

단점:
→ 정합장치 필요! ✅
→ 회로 복잡
```

**주로 사용:**
```
거리가 길 때
다중 주파수 사용 시
```

---

### 정합장치가 필요한 이유

```
비동조 급전선 사용 시
급전선 임피던스 ≠ 안테나 임피던스

임피던스 불일치
→ 반사파 발생
→ 정재파 생김
→ 비동조 급전선 의미 없음!

∴ 정합장치로
임피던스 일치시켜야 함 ✅
```

---

### 동조 vs 비동조 비교

| 구분 | 동조 급전선 | 비동조 급전선 |
|------|------|------|
| 정재파 | 있음 | 없음 |
| 정합장치 | 불필요 | **필요** |
| 전력손실 | 큼 | 적음 |
| 길이제한 | 있음(λ/2배수) | 없음 |
| 사용거리 | 짧을 때 | 길 때 |

---

### 핵심 암기

> **동조 급전선** = 정재파 있음 = 정합장치 불필요
> **비동조 급전선** = 정재파 없음 = 정합장치 **필요** ✅
>
> 헷갈리지 말것!
> 비동조가 오히려 정합장치 필요 ✅

## 왜 정재파 있으면 정합장치 불필요한가?
### 일반적 생각 vs 실제

```
일반적 생각:
정재파 있음 = 반사파 있음 = 나쁨
→ 정합장치 필요할 것 같음?

실제:
동조 급전선은 처음부터
정재파를 이용하는 방식! ✅
```

---

### 핵심 원리

**동조 급전선은 길이를 λ/2 배수로 맞춤**

```
λ/2 배수 길이
→ 임피던스 변환 효과 발생
→ 안테나 임피던스가
→ 송신기 쪽에서 보면 동일하게 보임!
```

---

### λ/2 임피던스 변환 원리

```
급전선 길이 = λ/2 일때

안테나쪽 임피던스 Z_A
       ↓
λ/2 선로를 통과하면
       ↓
송신기쪽에서 보이는 임피던스 = Z_A

→ 임피던스가 그대로 반복! ✅
```

그림으로:
```
송신기        λ/2 급전선        안테나
  Z_in ←──────────────────── Z_A

Z_in = Z_A
→ 자동으로 정합! ✅
→ 별도 정합장치 불필요!
```

---

### 비유로 이해

**거울을 두 번 보는 것과 같음:**

```
거울 1번 → 좌우 반전
거울 2번 → 다시 원래대로

λ/4 → 임피던스 반전
λ/2 → 다시 원래 임피던스로! ✅
```

---

### λ/4일 때는?

```
λ/4 선로:
Z_in = Z0² / Z_A

임피던스가 변환됨!
→ 정합장치 필요 ❌

λ/2 선로:
Z_in = Z_A

임피던스 그대로!
→ 정합장치 불필요 ✅
```

---

### 비동조 급전선과 비교

```
비동조 급전선:
길이가 λ/2 배수가 아님
→ 임피던스 변환 효과 없음
→ Z_in ≠ Z_A
→ 임피던스 불일치
→ 정합장치 필요! ✅

동조 급전선:
길이가 λ/2 배수
→ Z_in = Z_A 자동 성립
→ 정합장치 불필요! ✅
```

---

### 결론

> **동조 급전선 = λ/2 배수 길이**
>
> λ/2마다 임피던스가 똑같이 반복
> → 송신기-안테나 자동 정합
> → 정합장치 불필요 ✅
>
> 정재파가 있어서 불필요한게 아니라
> **λ/2 길이 덕분에 자동 정합!** ✅
