### 복사전계 vs 유도전계

**안테나 주변 전계 종류:**

```
안테나
  |
  |←── 근거리 ──→|←── 원거리 ──→
  |
  |  유도전계     복사전계
  |  (근접장)     (원거리장)
```

**유도전계 (Induction Field):**
```
거리에 따라 급격히 감소 (1/d²또는1/d³)
안테나 바로 근처에서 강함
에너지를 저장했다 돌려줌
실제 전파로 방사 안됨
```

**복사전계 (Radiation Field):**
```
거리에 따라 서서히 감소 (1/d)
먼 거리까지 전파됨
실제 전파 에너지
```

---

### 거리에 따른 세기 변화

```
세기
↑  유도전계(1/d²)
|  \
|   \     복사전계(1/d)
|    \   /
|     \ /
|      ×  ← 두 전계가 같아지는 지점
|       \  /
|        \/
└──────────────→ 거리
         d=λ/2π
```

---

### 근거리/원거리 경계

$$d = \frac{\lambda}{2\pi}$$

| 거리 | 지배적 전계 |
|------|------|
| $d < \frac{\lambda}{2\pi}$ | 유도전계 강함 (근접장) |
| $d = \frac{\lambda}{2\pi}$ | 두 전계 같음 |
| $d > \frac{\lambda}{2\pi}$ | 복사전계 강함 (원거리장) |

---

### 핵심 암기

> **복사전계 = 유도전계 되는 거리**
>
> $$d = \frac{\lambda}{2\pi}$$

## 반파장 안테나 전계강도
**자유공간에서 반파장 안테나 전계강도:**

$$E = \frac{7\sqrt{P}}{d} = \frac{60 I}{d} $$

- E : 전계강도 [V/m]
- P : 방사전력 [W]
- d : 거리 [m]

**상대이득 계산:**

$$G = 20\log\frac{E_{real}}{E_{ref}}$$


---

### 상대이득이란?

```
절대이득 → 등방성 안테나 기준
상대이득 → 반파장 안테나 기준
```

**비교:**
| 기준 안테나 | 이득 종류 | 단위 |
|------|------|------|
| 등방성 안테나 | 절대이득 | dBi |
| 반파장 안테나 | 상대이득 | dBd |

```
관계식:
dBi = dBd + 2.15
```
