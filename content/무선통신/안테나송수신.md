## 송신기 특성
주파수 안정도

안테나 공급 전력

변조 특성(직선성, 왜율, S/N비)

스퓨리어스 발사: 의도치 않은 주파수 신호 발생하는 것.

## 수신기 4대 지표
감도(Sensitivity): 아주 미약하거나 먼 거리의 전파를 수신할 수 있는 능력입니다. 감도가 좋을수록 더 작은 신호도 잘 잡아냅니다.

선택도(Selectivity): 수많은 전파 중에서 사용자가 원하는 특정 주파수의 신호만 정확하게 골라내고, 불필요한 잡음이나 혼신을 차단하는 능력입니다.

충실도(Fidelity): 수신한 신호(소리 등)를 원본에 얼마나 왜곡 없이 정확하게 재생해 내는가를 의미하는 지표입니다. (질문하신 '충성도'는 통상적으로 이 '충실도'를 의미합니다.)

안정도(Stability): 전원 전압의 변화나 주변 온도 변화, 장시간 작동에도 주파수가 변하지 않고 일정한 상태와 출력을 유지하는 능력입니다.

### SSB(Single Side Band)의 특징
* **스피치 크라리파이어(Speech Clarifier)**: 수신 측에서 반송파를 다시 삽입할 때, 미세한 주파수 차이를 조정하여 음성을 명료하게 만드는 장치입니다

### 수신기 방해 및 선택도
* **혼변조(Intermodulation)**: 통과대역 밖의 강력한 방해파가 희망파에 영향을 미치는 현상입니다
* **스켈치(Squelch)**: 무신호 시 들리는 불쾌한 잡음을 차단하는 기능입니다.


## 수신기의 4대 전기적 특성

### ① 감도 (Sensitivity)

```
수신기가 감지할 수 있는
최소 신호의 크기

약한 신호도 수신 가능 → 감도 좋음 ✅
```

> 멀리서 오는 약한 신호를
> 얼마나 잘 잡아내는가?

---

### ② 안정도 (Stability)

```
일정한 출력을
얼마나 오랫동안 유지하는가?

시간이 지나도 출력이 변하지 않으면
→ 안정도 높음 ✅
```

**안정도에 영향을 주는 요인:**
```
온도 변화
전원 전압 변동
부품 노화
진동/충격
```

---

### ③ 충실도 (Fidelity)

```
입력 신호를 얼마나
충실하게 재현하는가?

왜곡 없이 원음 그대로 재생
→ 충실도 높음 ✅
```

> 음악으로 비유하면
> 원음과 얼마나 똑같이 들리는가?

---

### ④ 선택도 (Selectivity)

```
원하는 주파수만 골라서
수신하는 능력

인접 채널 신호를 얼마나
잘 차단하는가?
```

**예시:**
```
라디오 채널 선택 시
옆 채널 신호가 섞이지 않으면
→ 선택도 높음 ✅
```

---

### 4대 특성 한눈에 비교

| 특성 | 의미 | 핵심 키워드 |
|------|------|------|
| **감도** | 최소 신호 감지 능력 | 약한 신호 |
| **안정도** | 출력 유지 시간 | 시간, 일정 출력 |
| **충실도** | 신호 재현 능력 | 왜곡 없음 |
| **선택도** | 원하는 주파수 선택 | 채널 분리 |

---

### 핵심 암기

> **안정도 = 시간이 지나도 출력이 일정**
>
> 감도 = 약한 신호도 OK
> 충실도 = 원음 그대로
> 선택도 = 원하는 채널만 ✅
