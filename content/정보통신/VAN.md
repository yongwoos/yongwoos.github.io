## 부가가치통신망(VAN)의 4계층 구조
VAN은 단순한 전송로 이상의 서비스를 제공하기 위해 다음과 같은 4개의 계층으로 구성됩니다.
```
정보처리 계층 (Information Processing Layer): 응용 소프트웨어를 통해 정보를 가공, 축적, 처리하는 최상위 계층입니다. 

통신처리 계층 (Communication Processing Layer): 서로 다른 기종 간의 통신이 가능하도록 프로토콜 변환, 부호 변환, 속도 변환 등을 수행하는 계층입니다. 

네트워크 계층 (Network Layer): 패킷 교환 방식 등을 통해 데이터를 전송하는 경로를 제어하고 중계하는 계층입니다. 

전송 계층 (Transmission Layer): 물리적인 통신 회선을 통해 데이터를 실제로 전송하는 하부 계층입니다.
```
