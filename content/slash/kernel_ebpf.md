### NUMA
- multi socket cpu
- 채널을 쪼개서 CPU별로 로컬메모리를 부착해 채널의 한계를 극복한 것
- local access와 remote access
- remote access
  - 비효율적으로 작동할 수 있음
  - local dram과 remote dram의 합의 40%가 remote access가 차지
- cpuset_cpu
- cpuset_memory
- 값을 조절해서 하드웨어 조정 가능
